# Product Hunt Asset Generator Instructions

## 1. Local Setup
Ensure your dev server is running:
```bash
npm run dev
```

## 2. Open the Generator
Navigate to: [http://localhost:3000/marketing-kit](http://localhost:3000/marketing-kit)

## 3. Generate High-Quality Assets

### Method A: One-Click Export (New!)
I have added a **"Download PNG"** button above each slide. 
1. Simply click the button.
2. It will automatically convert the slide into a high-resolution (2x Retina) PNG file.
3. The file will download as `slide-hero.png`, `slide-privacy.png`, etc.

### Method B: Browser DevTools (Backup)
If for some reason the auto-download doesn't render fonts correctly:
1. Press `F12` to open DevTools.
2. Toggle the **Device Toolbar** (`Ctrl+Shift+M`).
3. Set the dimensions to **1270 x 760**.
4. Capture the node screenshot via the command palette (`Ctrl+Shift+P`).

## 4. Mobile Editing (PixelLab)
Once you have the images:
1. Transfer them to your phone.
2. Open **PixelLab**.
3. Import the image.
4. Add any extra text or effects.
5. Save as **Ultra** quality.
